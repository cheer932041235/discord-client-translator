## 关键信息

| 项 | 值 |

| 项目类型 | 本地工具 / Discord 客户端 UI 增强 |

| 翻译模型 | DeepSeek 官方 API，默认 `deepseek-v4-flash` |

## 阻塞/风险

- Discord 客户端 Mod 存在 ToS 灰区，第一阶段仅用于观察号。
- 客户端插件生态需要调研，不能盲目选型。
- API Key 必须隔离在本地服务，不能写入客户端插件。
- 自动翻译容易造成 API 调用量过大，MVP 必须加入过滤、缓存、限流和去重。
